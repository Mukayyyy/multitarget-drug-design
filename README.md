# multitarget-drug-design

## Overview
This repository houses the code, data, and results for "Several Birds with One Stone: Exploring the Potential of AI Methods for Multi-Target Drug Design." The focus is on leveraging artificial intelligence to develop multi-target drug designs efficiently.

## Repository Contents
- **/models/**: Source code for each ML model used in experiments:
  - **/FragVAE_DEL**:
    - **/single_target**: Code and results for experiments targeting a single protein.
    - **/dual_target**: Code and results for experiments targeting two proteins.
    - **/triple_target**: Code and results for experiments targeting three proteins.
  - **/JTVAE_DEL**:
    - **/single_target**
    - **/dual_target**
    - **/triple_target**
  - **/AAE_DEL**:
    - **/single_target**
    - **/dual_target**
    - **/triple_target**
- **/data/**: Scripts to download and preprocess the dataset.
- **/docs/**: Supplementary documentation including extended analysis and figures.
